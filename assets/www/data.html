Работа с данными
Работа с данными в JsMobileBasic

localSaveData (name, _data)

Описание:
Сохранить данные в хранилище
Параметры:
Название Тип Описание
name string Название ячейки
_data * Данные

localReadData (name)

Описание:
Получить данные из хранилища
Параметры:
Название Тип Описание
name string Название ячейки

parseJSON (json)

Описание:
Возвращает объект из JSON строки
Параметры:
Название Тип Описание
json string JSON строка

toJSON (object, f opt, s opt)

Описание:
Возвращает JSON строку из объекта
Параметры:
Название Тип Атрибуты По умолчанию Описание
object object Объект
f function <опциональный>
null Дополнительный обработчик
s number <опциональный>
4 Отступ

toPSON (object, s opt)

Описание:
Возвращает PSON строку из объекта (с функциями)
Параметры:
Название Тип Атрибуты По умолчанию Описание
object object Объект
s number <опциональный>
4 Отступ
