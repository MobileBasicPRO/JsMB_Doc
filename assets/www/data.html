Работа с данными
Работа с данными в JsMobileBasic
toJSON(object) Описание:Конвертирует объект object в формат JSON. Параметры: object: объект, object; Возвращаемое значение: JSON строка Пример использования: a = toJSON({title:"test"}); // a = '{"title":"test"}' parseJSON(json) Описание:Парсит JSON строку json. Параметры: json: JSON строка, string Возвращаемое значение: объект Пример использования: a = parseJSON('{"title":"test"}'); //a = 5 parseJSON(json) Описание:Парсит JSON строку json. Параметры: json: JSON строка, string Возвращаемое значение: объект Пример использования: a = parseJSON('{"title":"test"}'); //a = 5
