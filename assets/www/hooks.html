О программе
Принимали участие в создании приложения:
Хуки (в программировании) - это обработчики событий.
В JsMB они пишутся так:
function onClick(x,y){
//Код } На сегодняшний день в JsMobileBasic доступны следующие хуки: onClick(x,y)
onRightClick(x,y)
onMouseDown(x,y)
onMouseUp(x,y)
onMouseMove(x,y)
onKeyPress(code)
onKeyDown(code)
onKeyUp(code)
 Теперь о каждом подробнее:
onClick(x,y) Когда срабатывает: При клике левой клавишей мыши Что возвращает: x - X координата мыши при клике, px
y - Y координата мыши при клике, px onRightClick(x,y) Когда срабатывает: При клике правой клавишей мыши Что возвращает: x - X координата мыши при клике, px
y - Y координата мыши при клике, px onMouseDown(x,y) Когда срабатывает: При нажатии любой клавиши мыши Что возвращает: x - X координата мыши при действии, px
y - Y координата мыши при действии, px onMouseUp(x,y) Когда срабатывает: При отжатии любой клавиши мыши Что возвращает: x - X координата мыши при действии, px
y - Y координата мыши при действии, px onMouseMove(x,y) Когда срабатывает: При движении мыши Что возвращает: x - X координата мыши при движении, px
y - Y координата мыши при движении, px onKeyDown(code) Когда срабатывает: При нажатии любой клавиши Что возвращает: code - код нажатой клавиши onKeyUp(code) Когда срабатывает: При отжатии любой клавиши Что возвращает: code - код отжатой клавиши onKeyPress(code) Когда срабатывает: Циклично при отжатии любой клавиши мыши Что возвращает: code - код зажатой клавиши
Примеры использования: function onKeyDown(code){ println(code);//Выводит на экран код нажатой клавиши для дальнейшего использования }
