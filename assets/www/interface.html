Операторы интерфейса
Операторы интерфейса:
menuAdd(menu, title, onclick) Описание Добавляет новый пункт в меню.
Если меню не существует - оно будет создано Параметры menu - название меню, строка
title - название элемента, строка
onclick - при нажатии, функция Возвращаемое значение: true Пример использования: menuAdd("main","Закрыть окно", Close);
