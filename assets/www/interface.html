Операторы интерфейса
Операторы интерфейса в JsMobileBasic
menuAdd(menu,title,onclick) Описание: Добавляет пункт title в меню menu; при нажатии выполнится функция onclick Параметры: menu: имя меню, string;
title: текст пункта меню, string
onclick: функция при нажатии, function
 Возвращаемое значение: true Пример использования: menuAdd("file","Открыть файл",open()); menuAddSeparator(menu) Описание: Добавляет разделитель в меню menu. Параметры: menu: имя меню, string;
 Возвращаемое значение: true Пример использования: menuAddSeparator("file"); menuShow(menu,x,y) Описание: Отображает меню menu по координатам x,y Параметры: menu: имя меню, string;
x,y: координаты левого верхнего угла меню, px
 Возвращаемое значение: true Пример использования: menuShow("file",100,100); menuBarAdd(bar,title,menu) Описание: Добавляет пункт в меню-бар bar при нажатии отобразится меню menu Параметры: bar: имя меню-бара, string;
title: текст пункта, function
menu: имя меню, string;
 Возвращаемое значение: true Пример использования: menuBarAdd("main","Файл","file"); menuBarShow(bar) Описание: Выводит на экран меню-бар bar в меню-бар bar Параметры: bar: имя меню-бара, string;
 Возвращаемое значение: true Пример использования: menuBarShow("main"); menuTrayAdd(menu) Описание: Добавляет меню в трей tray Параметры: menu: имя меню, string;
 Возвращаемое значение: true Пример использования: menuShow("file",100,100);
