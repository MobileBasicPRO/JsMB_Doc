Операторы ввода-вывода
Операторы ввода-вывода в JsMobileBasic

input (text, def opt)

Описание:
Окно ввода данных
Параметры:
Название Тип Атрибуты Описание
text string Текст заголовка окна
def string <опциональный>
Текст по умолчанию

println ()

Описание:
Вывести текст на экран

log ()

Описание:
Логирование
Параметры:
Тип Описание
* Данные

debug (text, style opt)

Описание:
Вывести сообщение для отладки
Параметры:
Название Тип Атрибуты Описание
text string Сообщение
style string <опциональный>
Оформление сообщения (CSS)
