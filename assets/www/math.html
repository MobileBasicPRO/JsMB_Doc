Математические функции
Графические операторы в JsMobileBasic
sin(x) Описание: Возвращает синус числа x. Параметры: x: величина угла в радианах (рад.). Возвращаемое значение: синус угла x. Пример использования: a = sin(0); //a = 0 cos(x) Описание: Возвращает косинус числа x. Параметры: x: величина угла в радианах (рад.). Возвращаемое значение: косинус угла x. Пример использования: a = cos(0); //a = 1 tan(x) Описание: Возвращает тангенс числа x. Параметры: x: величина угла в радианах (рад.). Возвращаемое значение: тангенс угла x. Пример использования: a = tan(0); //a = 0 ctg(x) Описание: Возвращает котангенс числа x. Параметры: x: величина угла в радианах (рад.). Возвращаемое значение: котангенс угла x. Пример использования: a = ctg(1); //a = 0.6420926159343307 mod(x, y) Описание: возвращает остаток от целочисленного деления числа x на число y. Параметры: x, y: числа. Возвращаемое значение: остаток от целочисленного деления числа x на число y. Пример использования: a = mod(5.7,2.5);//a = 0.7 abs(x) Описание: возвращает значение модуля числа x. Параметры: x: число. Возвращаемое значение: модуль числа x. Пример использования: a = abs(-5);//a=5 sqrt(x) Описание: возвращает значение корня квадартного числа x. Параметры: x: число. Возвращаемое значение: корень квадартный числа x. Пример использования: a = sqrt(9);//a=3 random(a,b) Описание: возвращает псевдослучайное число в диапазоне от a до b. Параметры: a, b: числа. Возвращаемое значение: псевдослучайное число в диапазоне от a до b. Пример использования: r = random(0,255); g = random(0,255); b = random(0,255); fillScreen('rgb('+r+','+g+','+b+')'); pow(x, num) Описание: возводит число x в степень num. Параметры: x, num: основание степени, показатель степени. Возвращаемое значение:результат возведения числа n в степень num. Пример использования: a = pow(2, 8);//a=256 asin(x) Описание: возвращает значение арксинуса числа x в радианах (рад.). Параметры: x: значение синуса искомого угла. Возвращаемое значение: арксинус числа x. Пример использования: a = asin(0);//a=0 acos(x) Описание: возвращает значение арккосинуса числа x в радианах (рад.). Параметры: x: значение косинуса искомого угла. Возвращаемое значение: арккосинус числа x. Пример использования: a = acos(1);//a=0 atan(x) Описание: возвращает значение арктангенса числа x в радианах (рад.). Параметры: x: значение тангенса искомого угла. Возвращаемое значение: арктангенс числа x. Пример использования: a = atan(0);//a=0
