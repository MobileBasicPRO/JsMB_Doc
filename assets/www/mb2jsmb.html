С Mobile Basic в JsMobileBasic
Миграция с JsMobileBasic:
Переменные Mobile Basic: A%=5;
A$="String";
A=5.5; JsMobileBasic: a=5;
a="String";
a=5.5; Условия Mobile Basic: IF A%=5 THEN CLS JsMobileBasic: if(a == 5){ cls(); }; Mobile Basic: IF A%=1 THEN PRINT "true" IF A%=0 THEN PRINT "false" JsMobileBasic: if(a){ println("true"); }else{ println("false"); } Циклы Mobile Basic: FOR I%=0 TO 100 STEP 5 JsMobileBasic: for(i=0;i<100;i+=5) Обработка событий

Так, как в Mobile Basic была поддержка только клавиатуры телефона и использовалась простая система событий в виде функций, которые возвращали 1 при событии (FIRE(0)).
JsMobileBasic поддерживает все клавиши мыши и клавиатуры, по этому, для вашего удобства, мы изменили её.

MobileBasic: IF FIRE(0) THEN ... JsMobileBasic: function onKeyDown(code){ if(code == 5){//Если нажат Enter //... } }
