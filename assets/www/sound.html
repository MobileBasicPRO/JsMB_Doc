Работа со звуком
Работа со звуком в JsMobileBasic

playSound (file, loop, channel)

Описание:
Играть звук
Параметры:
Название Тип По умолчанию Описание
file string Файл звука
loop bool Зациклить?
channel string 0 Канал

pauseSound (channel)

Описание:
Приостановить воспроизведение звука на канале
Параметры:
Название Тип По умолчанию Описание
channel number -1 Канал (-1 для остановки на всех каналах)
