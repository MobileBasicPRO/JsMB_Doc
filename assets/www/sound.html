Работа со звуком
Работа со звуком в JsMobileBasic
playSound(file,loop, channel) Описание:Проигрывает звук file на канале channel. Параметры: file: путь к файлу (поддерживает http), строка;
loop: повторение, bool;
channel: канал, number; Возвращаемое значение: true Пример использования: playSound("music.mp3", true, 1); pauseSound(channel) Описание:Останавливает звук на канале channel. Параметры: channel: канал, number Возвращаемое значение: число из строки str, или NaN. Пример использования: a = int("5"); //a = 5
a = val("100"); //a = 100
