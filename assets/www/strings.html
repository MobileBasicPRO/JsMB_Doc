Строковые функции
Строковые функции в JsMobileBasic
len(str) Описание: Возвращает длину строки str. Параметры: str: Строка. Возвращаемое значение: длина строки str. Пример использования: a = len("Hello world"); //a = 11 int(str)
val(str) Описание: Парсит число из строки str. Параметры: str: Строка. Возвращаемое значение: число из строки str, или NaN. Пример использования: a = int("5"); //a = 5
a = val("100"); //a = 100 float(str) Описание: Парсит дробь из строки str. Параметры: str: Строка. Возвращаемое значение: десятичная дробь из строки str. Пример использования: a = float("10.54"); //a = 10.54 upper(str) Описание:Переводит строку str в верхний регистр. Параметры: str: Строка. Возвращаемое значение: строка в верхнем регистре. Пример использования: a = upper("word"); //a = "WORD" lower(str) Описание: Переводит строку str в нижний регистр. Параметры: x, y: числа. Возвращаемое значение:строка в нижнем регистре. Пример использования: a = lower("word");//a = "WORD" split(str, sym) Описание: Разбивает строку str по символам sym. Параметры: str: Строка.
sym: Символ, строка Возвращаемое значение: массив символов. Пример использования: a = split("hello","");//a = ["h","e","l","l","o"] replace(str,regexp,to) Описание: Заменяет в строке символы по регулярному выражению regexp на to. Параметры: str: Строка;
regexp: Регулярное выражение;
to: Замена, строка Возвращаемое значение: строка Пример использования: a = replace("Hello world!",/world/g,"me");//a = "Hello me!" mid(str, start, end) Описание: Вырезает строку в диапазоне от start до end. Параметры: str: строка;
start, end: числа. Возвращаемое значение: строка. Пример использования: a=("world",0,2); //a = "wo" char(sym) Описание: Возвращает код символа sym. Параметры: sym: символ, строка. Возвращаемое значение:код символа sym. Пример использования: a = char("h");//a = 256 asc(char) Описание: Bозвращает символ за его значением char. Параметры: char: код символа, строка. Возвращаемое значение: Символ. Пример использования: a = asc("228");//a = ""
