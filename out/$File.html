Префикс: $File

$File

Модуль для работы с файлами Поддерживает: JsMB-Interpreter NW.JS Node.JS JsOS
Автор:
  • PROPHESSOR

Операторы и функции

saveData(filename, data)

Описание:
Сохраняет данные в папку приложения (appdata)
Параметры:
Название Тип Описание
filename string Имя файла
data object Данные для сохранения
Возвращаемое значение:
Тип: this

readData(filename)

Описание:
Читает данные из файла в папке приложения (appdata)
Параметры:
Название Тип Описание
filename string Имя файла
Возвращаемое значение:
Объект или строка
Тип: object

readFile(file)

Описание:
Прочитать данные из файла
Параметры:
Название Тип Описание
file string Файл
Возвращаемое значение:
Тип: string

writeFile(file, data)

Описание:
Записать в файл
Параметры:
Название Тип Описание
file string Файл
data string Данные
Возвращаемое значение:
Тип: this

readFileBinary(file)

Описание:
Чтение файла в бинароном виде
Параметры:
Название Тип Описание
file string Файл
Возвращаемое значение:
Тип: Buffer

writeFileBinary(file, data)

Описание:
Запись файла в бинарном виде
Параметры:
Название Тип Описание
file string Файл
data Buffer Буфер с данными
Возвращаемое значение:
Тип: this
