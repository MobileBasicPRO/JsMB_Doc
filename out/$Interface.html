Префикс: $Interface

$Interface

Модуль для работы с интерфейсом Поддерживает только JsMB-Interpreter и NW.JS
Автор:
  • PROPHESOR

Операторы и функции

menuAdd(name, title, onClick, typeopt, fortypeopt)

Описание:
Добавить пункт в контекстное меню
Параметры:
Название Тип Атрибуты Описание
name string ID название меню
title string Текст пункта меню
onClick function Функция выполнится при нажатии на пункт меню
type string <опциональный>
 Тип меню (subMenu, checkbox, separator)
fortype string <опциональный>
 Родительское меню для subMenu
Возвращаемое значение:
Тип: this
